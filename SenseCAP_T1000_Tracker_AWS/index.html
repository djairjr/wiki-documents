Create the AWS Account

Go to https://signin.aws.amazon.com/ to create an AWS account.

This will need your email, name, credit card info.

after the registration, you are ready to go to AWS IoT in the console.

Add Gateway‚Äã

On the left menu,select LPWAN devices ‚Üí Gateways

you can use the SenseCAP M2 gateway

SenseCAP M2 Multi-Platform LoRaWAN Indoor Gateway(SX1302)¬†-¬†US915

click add Gateway

Input the Gateway EUI which you can get it in the Gateway admin console

set the frequency plan, US915 for United States and set the sub band as 2

next, create the certificate, which will be need in the gateway configuration. with this cerificate, the SenseCAP M2 can connect to AWS IoT Core LoRaWAN.

download and save the certificate files and the server trust certificates.

unzip it, there are four files inside.

next create a role fro the gateway.

copy the CUPs url, this is important, which tells the gateway where the data should be sent to.

config the SenseCAP M2 with the right selection, and certificate from those downloaded files.

now, you will see the Gateway connected with AWS

Add Devices‚Äã

add SenseCAP Tracker to AWS IoT Core

first, you need to add device profile

and create the device service profile

then add desination, which will tell where the data should go.

next, add the detail SenseCAP Tracker T1000

read out the device EUI , Appkey, and AppEUI from the SenseCAP T1000 from the SenseCAP Mate App

and paste it in the AWS

select the device profile just created

and select the destination just configured

untill now, the device has been added to AWS IoT Core.

Decode the Tracker data‚Äã

The SenseCAP Tracker will uplink the payload via LoRaWAN networkÔºåand the payload is under specified format to save the bandwith, so it is hard to read.

So we need to decode it, and save the data somewhere in AWS.

we need to create a rules for the Tracker.

For prepare, you need to record the ‚ÄúEndpoint" first, go to AWS IoT core home page, and click ‚ÄúSettings‚Äù

Record the ‚ÄúEndpoint‚Äù address in the page, you will need to use it in the next step.

For example, what I've shown here is a39w0g3w5os1ti-ats.iot.us-east-1.amazonaws.com.

Create Message Rules‚Äã

Click Message routing ‚Üí Rules, and click ‚ÄúCreate Rule‚Äù button.

Input a rule name(‚Äúsensecap_lorawan_rule‚Äù for example), and then click ‚ÄúNext‚Äù.

Select ‚Äú2016-03-23‚Äù from SQL version, and input the following content, as the contents of ‚ÄúSQL statement‚Äù:

SELECT * FROM ‚Äúlorawan-device‚Äù

Same page, scroll down to ‚ÄúRule actions‚Äù section, and select ‚ÄúLambda‚Äù from ‚ÄúAction 1‚Äù.

Next is an empty drop-down menu for the ‚ÄúLambda function‚Äù, we need to create a function first, click ‚ÄúCreate a Lambda function‚Äù and jump to another page.

Refer to the screenshot below, input ‚ÄúFunction name‚Äù and select the correct ‚ÄúRuntime‚Äù, and then, click ‚ÄúCreate function‚Äù button to create a new function.

On the following funcition config page, remove all the code and replace it with the script attached at the end of the guide, then click ‚ÄúDeploy‚Äù button.

In the third line of the script, you need to replace the address in single quotes with the Endpoint address you recorded earlier.

Click ‚ÄúConfiguration‚Äù ‚Üí ‚ÄúPermissions‚Äù ‚Üí ‚ÄúEdit‚Äù

Click "View the xxxxxxxxxxx role" at the bottom.

Click ‚ÄúAdd permissions‚Äù ‚Üí ‚ÄúAttach policies‚Äù.

Search ‚ÄúAdministratorAccess‚Äù, check the box left it, and then click ‚ÄúAdd Permissions‚Äù.

Go back to ‚ÄúCreate rule‚Äù page, click refresh button and select the lambda function you created.

Then, click ‚ÄúCreate‚Äù and complete the rule creation.

Watch the data on page ‚ÄúMQTT test client‚Äù, input # and click ‚ÄúSubscribe‚Äù button, you will see the tracker‚Äôs data.

    function decodeUplink (input) {
        // var bytes = input['bytes']
        var bytes = Buffer.from(input, 'base64');
        var bytesString = bytes2HexString(bytes).toLocaleUpperCase()
        var decoded = {
            valid: true,
            err: 0,
            payload: bytesString,
            messages: []
        }
        let measurement = messageAnalyzed(bytesString)
        decoded.messages = measurement
        return { data: decoded }
    }
    
    function messageAnalyzed (messageValue) {
        try {
            let frames = unpack(messageValue)
            let measurementResultArray = []
            for (let i = 0; i < frames.length; i++) {
                let item = frames[i]
                let dataId = item.dataId
                let dataValue = item.dataValue
                let measurementArray = deserialize(dataId, dataValue)
                measurementResultArray.push(measurementArray)
            }
            return measurementResultArray
        } catch (e) {
            return e.toString()
        }
    }

    function bytes2HexString (arrBytes) {
        var str = ''
        for (var i = 0; i < arrBytes.length; i++) {
            var tmp
            var num = arrBytes[i]
            if (num < 0) {
                tmp = (255 + num + 1).toString(16)
            } else {
                tmp = num.toString(16)
            }
            if (tmp.length === 1) {
                tmp = '0' + tmp
            }
            str += tmp
        }
        return str
    }
    function loraWANV2DataFormat (str, divisor = 1) {
        let strReverse = bigEndianTransform(str)
        let str2 = toBinary(strReverse)
        if (str2.substring(0, 1) === '1') {
            let arr = str2.split('')
            let reverseArr = arr.map((item) => {
                if (parseInt(item) === 1) {
                    return 0
                } else {
                    return 1
                }
            })
            str2 = parseInt(reverseArr.join(''), 2) + 1
            return '-' + str2 / divisor
        }
        return parseInt(str2, 2) / divisor
    }
    
    function bigEndianTransform (data) {
        let dataArray = []
        for (let i = 0; i < data.length; i += 2) {
            dataArray.push(data.substring(i, i + 2))
        }
        return dataArray
    }
    
    function toBinary (arr) {
        let binaryData = arr.map((item) => {
            let data = parseInt(item, 16)
                .toString(2)
            let dataLength = data.length
            if (data.length !== 8) {
                for (let i = 0; i < 8 - dataLength; i++) {
                    data = `0` + data
                }
            }
            return data
        })
        return binaryData.toString().replace(/,/g, '')
    }

    function getMacAndRssiObj (pair) {
        let pairs = []
        if (pair.length % 14 === 0) {
            for (let i = 0; i < pair.length; i += 14) {
                let mac = getMacAddress(pair.substring(i, i + 12))
                if (mac) {
                    let rssi = getInt8RSSI(pair.substring(i + 12, i + 14))
                    pairs.push({mac: mac, rssi: rssi})
                } else {
                    continue
                }
            }
        }
        return pairs
    }
    
    function getMacAddress (str) {
        if (str.toLowerCase() === 'ffffffffffff') {
            return null
        }
        let macArr = []
        for (let i = 1; i < str.length; i++) {
            if (i % 2 === 1) {
                macArr.push(str.substring(i - 1, i + 1))
            }
        }
        let mac = ''
        for (let i = 0; i < macArr.length; i++) {
            mac = mac + macArr[i]
            if (i < macArr.length - 1) {
                mac = mac + ':'
            }
        }
        return mac
    }
    
    function getInt8RSSI (str) {

    function getEventStatus (str) {
        let bitStr = getByteArray(str)
        let event = []
        for (let i = bitStr.length; i >= 0; i--) {
            if (i === 0) {
                event[i] = bitStr.substring(0)
            } else {
                event[i] = bitStr.substring(i - 1, i)
            }
        }
        return event.reverse()
    }
    
    function getByteArray (str) {
        let bytes = []
        for (let i = 0; i < str.length; i += 2) {
            bytes.push(str.substring(i, i + 2))
        }
        return toBinary(bytes)
    }
    
    function getWorkingMode (workingMode) {
        return getInt(workingMode)
    }
    
    function getUTCTimestamp(str){
        return parseInt(loraWANV2PositiveDataFormat(str)) * 1000
    }
    
    function loraWANV2PositiveDataFormat (str, divisor = 1) {
        let strReverse = bigEndianTransform(str)
        let str2 = toBinary(strReverse)
        return parseInt(str2, 2) / divisor
    }
    
    exports.handler = async (event) => {
      try {
        const lorawan_info = event["WirelessMetadata"]["LoRaWAN"];
        const lorawan_data = event["PayloadData"];
    
        const params = {
          topic: 'sensecap/t1000',
          payload: JSON.stringify({
            eui: lorawan_info["DevEui"],
            timestamp:  lorawan_info["Timestamp"],
            data: decodeUplink(lorawan_data)["data"]
          }),
          qos: 0
        };
    
        await iotdata.publish(params).promise();
    
        return {
          statusCode: 200,
          body: 'Message published successfully'+JSON.stringify(event)
        };
      } catch (error) {
        console.error('Error publishing message:', error);
    
        return {
          statusCode: 500,
          body: 'Error publishing message'
        };
      }
    };
